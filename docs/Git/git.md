# Git学习笔记

## Git Bash下安装tree

- Windows
    1. 下载[Tree for Windows](http://gnuwin32.sourceforge.net/packages/tree.htm)，选择 Binaries 类型的 zip 包。
    2. 将下载的 zip文件（`tree-1.5.2.2-bin.zip` ）中 `bin/tree.exe` 文件复制到 Git Bash 安装目录（如：`C:\Program Files\Git\usr\bin`）下。
