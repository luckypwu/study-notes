### 配置

1. 配置工作区路径，参考[链接](https://blog.csdn.net/weixin_45312236/article/details/137077440)。

    ```json
    
    {
      // 省略其他
      "terminal.integrated.env.windows": {
        "PYTHONPATH": "${workspaceFolder};${env:PYTHONPATH}"
      }
    }
    ```

    注意：配置完成后，需要重启 VS Code 才会生效。
